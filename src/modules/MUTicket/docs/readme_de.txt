INSTALLATIONSANLEITUNG
----------------------
1) Kopieren Sie MUTicket in das Verzeichnis modules.
2) Initialisieren und aktivieren Sie MUTicket in der Modulverwaltung.
3) Setzen Sie das Verzeichnis `userdata/MUTicket/` inklusive aller Unterordner auf beschreibbar.
